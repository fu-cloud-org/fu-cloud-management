## 项目介绍

福云后台管理系统是基于vue-element-template脚手架搭建的”福云论坛“网站综合管理平台，在这里提供了以下功能：

​	1.图表项数据

​	2.用户管理

​	3.帖子管理

​	4.标签管理

​	5.推广链接管理

[”福云“论坛网站在这](120.76.202.109)

## 安装方法

```
# enter the project directory
cd 文件所在目录

# install dependency
npm install

# develop
npm run dev
```
